How does bubble sort work? Click me to find out

If you are on mobile then click arrows in order to show next steps

First we have to create an array of numbers, for example 1, 3, 4, 2

Then we check which of two adjecent numbers starting from left is greater 1, 3, 4, 2

If it is smaller, then we leave it as it is and go to the next pair of numbers to compare so this time we compare 1, 3, 4, 2

3 is smaller than 4 so we leave it as it is and go to the next pair

If it is greater like in this example 1, 3, 4, 2

We just swap their positions and start again from the start with an array looking like this 1, 3, 2, 4

The process is repeated, so we compare 1, 3, 2, 4

We already know that 1 is smaller than 3 so we go to the next pair and compare 1, 3, 2, 4

3 is greater than 2 so we swap them around 1, 2, 3, 4

And 3 is smaller than 4 so we leave it as it is, and by doing so we sorted our array 1, 2, 3, 4 !!!
